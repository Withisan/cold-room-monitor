วิธีสมัคร Firebase ฟรี (คลิกดู)
1. เปิด console.firebase.google.com → Login Google
2. กด Add project → ตั้งชื่อ → สร้าง
3. เลือก Web app → Copy firebaseConfig
4. ไปที่ Firestore Database → Create database → Start in test mode
5. ไปที่ Authentication → Sign-in method → Email/Password → Enable
6. Authentication → Users → Add user → ใส่ Email+Password ของ Admin
7. ใส่ค่า config ด้านบน → กด เชื่อมต่อ Firebase

1. สมัคร emailjs.com → Add Email Service → Copy Service ID
2. Create Template → ใส่ตัวแปร: {{room_name}} {{alarm_type}} {{message}} {{temperature}} {{min_temp}} {{max_temp}} {{time}} {{to_email}} {{to_name}}
3. Account → General → Copy Public Key
4. Account → Security → Allowed Origins → Add → ใส่ URL GitHub Pages
