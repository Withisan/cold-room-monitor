📖 วิธีสมัคร EmailJS และตั้งค่า (คลิกเพื่อดู)
1สมัคร EmailJS — เปิด emailjs.com → Sign Up ฟรี (ใช้ Email บริษัทได้)

2เพิ่ม Email Service — ไปที่ Email Services → Add New Service → เลือก Gmail หรือ Outlook → Connect Account → Copy Service ID

3สร้าง Template — ไปที่ Email Templates → Create New Template
ใส่เนื้อหา template ดังนี้:
Subject: ⚠️ Cold Room Alert — {{room_name}}
Body:
เวลา: {{time}}
ห้อง: {{room_name}}
ประเภท: {{alarm_type}}
รายละเอียด: {{message}}
อุณหภูมิปัจจุบัน: {{temperature}}°C
ค่าที่กำหนด: Min {{min_temp}}°C / Max {{max_temp}}°C
Copy Template ID

4Copy Public Key — ไปที่ Account → General → Copy Public Key

5ใส่ค่าทั้ง 3 ในช่องด้านบน แล้วกด "ส่งทดสอบ"
⚙ ตั้งค่า Alarm บันทึกอัตโนมัติ
ระบบแจ้งเตือนเมื่อ:
🌡 อุณหภูมิสูงเกิน Max → ส่ง Email + เสียง ทันที
❄ อุณหภูมิต่ำกว่า Min (อุณหภูมิตก) → ส่ง Email + เสียง ทันที
🚪 ประตูเปิด → ส่ง Email + เสียง ทันที
🔄 ประตูเปิด/ปิด (เปลี่ยนสถานะ) → บันทึก log เท่านั้น

รูปแบบ Email ที่ส่ง:
Subject: ⚠️ Cold Room Alert — ห้องเย็น A
เวลา: 14:32:05
ห้อง: ห้องเย็น A
ประเภท: อุณหภูมิสูงเกิน
รายละเอียด: อุณหภูมิสูงเกิน -10.5°C (ค่าสูงสุด -15°C)
อุณหภูมิปัจจุบัน: -10.5°C
ค่าที่กำหนด: Min -20°C / Max -15°C
ความถี่: ส่งทันทีเมื่อเกิด alarm จากนั้น cooldown ตามที่ตั้งไว้เพื่อป้องกัน spam
Cooldown: นาที / ครั้ง (ป้องกัน Email ซ้ำบ่อยเกินไป)
ห้อง ชื่อห้อง Zone Target °C Min °C Max °C Warn ±°C
* Warn = ระยะห่างจาก Max/Min ที่แสดงสีเหลือง "เฝ้าระวัง" (ยังไม่ส่ง Email)
